CURL使用

  • CURL

CURL

  • -i 返回HTTP头

  • -H 设置请求头，例如curl URL -H 'Content-Type: application.json'

  • -d 后面添加POST内容，例如curl -XPOST URL -d "param1=value1&param2=value2"。

    JSON内容可以用引号括起来。例如：

    curl -XPOST 'example.com/info' -H 'Content-Type:application/json' -d '
{
    "name": "需要的是啥",
    "type": "question"
}
'

  • --data-urlencode 则对POST参数做转义，curl -XPOST --data-urlencode "param1=value1&param2=需要转义"

  • -G 发起GET请求

  • -F 上传文件，用法：curl URL -F "file=@**.doc" -F "name=filename"

注意点：

  • 通过双引号防止URL转义
